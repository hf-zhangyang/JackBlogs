# 入门指南

欢迎来到我的技术博客！这里提供了一些基础指南帮助你更好地使用本站。

## 内容导航

本博客分为几个主要部分：

1. **技术指南**：各种技术的入门教程和最佳实践
2. **博客文章**：原创技术文章和学习笔记
3. **关于我**：个人信息和联系方式

## 开始使用

### 本地阅读

你可以直接访问网站阅读所有内容，所有文章都在侧边栏中分类整理。

### 本地搭建

如果你想自己搭建类似的博客，可以参考：

1. VitePress 官方文档：[https://vitepress.dev/](https://vitepress.dev/)
2. 本项目的 GitHub 仓库：[https://github.com/hf-zhangyang/my-tech-blog](https://github.com/hf-zhangyang/my-tech-blog)

## 技术栈

本博客使用以下技术构建：

- **VitePress**：静态网站生成器
- **Vue 3**：组件开发
- **GitHub Pages**：免费托管
